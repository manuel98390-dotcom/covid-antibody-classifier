# Dataset (Public)

Este proyecto utiliza fuentes públicas de anticuerpos.

## 1) OAS (Observed Antibody Space)
Repositorio/portal usado para anticuerpos **no relacionados con SARS-CoV-2** (como negativos/contraste o datasets generales).

Links: https://opig.stats.ox.ac.uk/webapps/oas/

## 2) CoV-AbDab
Repositorio/portal usado para anticuerpos **relacionados con SARS-CoV-2**.

Links: https://opig.stats.ox.ac.uk/webapps/covabdab/ 

## Nota de reproducibilidad
- Este repositorio incluye un archivo pequeño de ejemplo en `data/sample/` para que el pipeline corra de inmediato.
- Para experimentos completos, se recomienda descargar/curar los datos desde las fuentes oficiales anteriores, respetando los términos de uso.
